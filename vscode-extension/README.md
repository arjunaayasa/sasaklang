SasakLang adalah bahasa pemrograman esoterik yang menggunakan kosa kata Bahasa Sasak.

## Fitur
- **Keyword Sasak**: `gawe`, `lamun`, `selame`, `fungsi`, dll.
- **Syntax Highlighting**.
- **Snippets**.

## Contoh
```sasak
cetak("Halo Dunia!")
```
